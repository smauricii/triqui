Descripcion
triqui es un juego donde 2 jugadores "O","X",
marcan los espacios de un tablero de 3×3 alternadamente.

Como se realizó
Principalmente para desarrollar este juego,
se utilizaron diferentes tecnologias que son:

°HTML: como vista
°CSS: diseño de la vista
°JavaScript: desarrollo del codigo

Aplicando diferentes metodos para que la
comunicacion entre la vista y el codigo sea correcta

Funcionalidad Adicional

se agregó un boton de reiniciar para no tener
que cargar la pagina repetidas veces para
volver a jugar
